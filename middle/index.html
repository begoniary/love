# 初中主线

# tr的第一印象

那是正衡去报道的第一天，早上出发的时候，心里有些忐忑，想着要早点去找个好位置，但是也不是很有所谓坐在哪里。虽然正衡常去，但是第一次以初中生的身份走进去，不知道会遇到什么样的新同学，也不知道会有什么样的新生活（？）

走进教室的时候，因为我妈告诉我要找trq坐在一起，是已经约好的，所以我开始四处张望，可惜举目无亲，但是我看见了一个认识的女生，是jjn，也不记得是怎么认识的，大概是家长认识我妈然后有过一面之缘吧。然后，理所应当的，我就注意到了坐她边上的小女生，怎么可能注意不到呢，当时她侧过身来，睁大了眼睛，双手在空中比划着些什么，或许算一见钟情，或许也不算，我只觉得这个女生好可爱。当时本以为本人出发早，到的已经很早了，结果没想到，到的时候，教室里居然已经坐了一多半的人了。我看了一圈也没有找到trq，随便找了个比较靠前，但也不算太前的位置坐下了，等trq来坐我的边上。在等的过程中，我开始打量着身边的同学们，但是一直吸引着我目光的还是刚刚看到的那个女生。我开始偷偷地看她，也不知道是不是刻意想看她，我的眼角总是悄悄地扫过那一束马尾辫。那天的zxy，戴着蓝色的头箍，梳着马尾辫，眼睛好大好大，而我还不知道她叫zxy。

# tr的第二印象

第二印象的时候已经知道她的名字，而且很奇怪的是，第二印象是看的她的照片，当时雷哥为了让我们早点互相认识，要我们每个人贴一张照片在教室后面的照片墙上，每个课间都赶我们去教室后头参观那面照片墙。我还记得，当时我自己放的是一张去青岛旅游的照片，在五四广场，笑的好勉强。唔但是当我看到zxy在照片墙上的照片的时候，突然有一种似曾相识的感觉，或许真的在哪里见过吧，但是我认识她的感受真的非常非常强烈。照片里有假山和小池塘，她站在池塘边的长排凳子上，笑的很可爱。那时，我突然意识到她好好看，好想多看看她。

然后就是同一个课间，我又去看了教室后面另一边的小报墙，唔是开学时没人做一张的小报，大概是自定的主题，然后选出了做的好的展示吧，反正我的小报不在上面。然后看到了一个做的很好看的黑白小报，现在想来有一点好笑，我当时真的很喜欢那首“滚滚长江东逝水”的《临江仙》，那张讲三国演义的小报最上面就是这首词，那我的目光不由自主地被它吸引啦，仔细看完感觉这个小报的作者写的好棒，本人试图找到这个作者的名字，可是小报的正面没有名字，小报的背面被大头钉定在了板上，我小心翼翼地拔起右下角的大头钉，当看到那三个字的时候，又匆匆忙忙，慌不择路地把大头钉重新钉上，装作什么都没有看到的样子，实际上心里已经波涛纵横。
